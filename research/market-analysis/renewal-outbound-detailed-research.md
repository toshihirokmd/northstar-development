# 更新アウトバウンド代行サービス 詳細マーケットリサーチ

**調査日**: 2026-03-11
**対象**: 継続率改善・更新アウトバウンド代行サービス

## 🎯 ターゲット業界分析

### 1️⃣ フィットネス業界（最有力ターゲット）

#### 継続率の深刻な課題
- **半年継続率**: わずか30%（70%が半年以内に退会）
- **1年継続率**: 約10%（90%が1年以内に退会）
- **24時間ジム**: より継続率改善の必要性が高い

#### 市場機会
- **改善の余地が巨大**: 30% → 50%改善でも大きなインパクト
- **解決策が明確**: 手厚いサポート・プログラムで改善可能
- **年間パス・月会費**: 継続的収益モデル

#### 具体的課題
- 入会後3ヶ月〜半年の離脱防止
- 孤独感・疎外感の解消
- 来店習慣の定着化
- インストラクターとの関係構築

### 2️⃣ 保険業界（安定ターゲット）

#### 継続率の現状
- **自動車保険**: 95.2%（東京海上日動）
- **生命保険**: IQA24ヶ月維持率95.9%、60ヶ月維持率86.0%
- **課題**: 保険料未収・契約失効顧客対応

#### 市場機会
- **入金約束率92%実績**: アウトバウンドの効果実証済み
- **大きな市場規模**: 保険料未収・失効顧客は相当数存在
- **高単価**: 1件の継続で年間数万〜数十万円の保険料

#### 具体的課題
- 保険料未収顧客への支払い勧奨
- 契約失効防止
- 更新時期の積極的フォロー

### 3️⃣ SaaS業界（成長ターゲット）

#### 継続率の現状
- **平均チャーンレート**: 月間3-10%
- **目標**: 月間3%以内
- **年間換算**: 良好企業でも15-20%は解約

#### 市場機会
- **急成長市場**: SaaS業界全体の拡大
- **高単価**: 企業向けSaaSは月数万〜数百万円
- **データ活用可能**: 利用状況データで予兆検知
- **カスタマーサクセス需要**: 専門性の高い領域

#### 具体的課題
- 利用開始後の定着支援
- 更新時期の価値再確認
- アップセル・クロスセルの機会創出

## 💰 料金体系・市場価格

### アウトバウンド代行の一般的料金体系

#### 成果報酬型
- **継続獲得**: 継続による年間売上の15-30%
- **更新成功**: 更新金額の10-25%
- **アポ獲得**: 1件あたり5,000-20,000円

#### 固定費 + 成果報酬型
- **基本料金**: 月額10-50万円
- **成果報酬**: 上記の50-70%

#### 従量課金型
- **コール単価**: 1コール200-500円
- **時間単価**: 1時間2,000-4,000円

### 競合企業の価格帯
- **大手企業**: 高品質・高価格（月額50-200万円+成果報酬）
- **中堅企業**: バランス型（月額20-100万円+成果報酬）
- **小規模企業**: 低価格特化（月額5-30万円+成果報酬）

## 🎯 さくらフォレストの参入戦略

### Phase 1: 九州・福岡特化（3-6ヶ月）

#### ターゲット
- **フィットネス業界**: 継続率30% → 50%改善
- **地域特化**: 九州・福岡エリアの中小ジム
- **開始規模**: 5-10社から

#### サービス内容
```
基本パッケージ：
- 入会後1ヶ月、3ヶ月、半年のタイミングでアウトバウンド
- 個別の課題ヒアリング・モチベーション維持
- 来店促進・プログラム提案
- インストラクターとの橋渡し

料金設定：
- 基本料金：月額10万円（5社まで）
- 成果報酬：継続による年間売上の20%
- 最低契約：6ヶ月
```

#### 期待される成果
```
例：会員500名、月額8,000円のジムの場合
継続率30% → 50%改善 = 100名増
年間売上増：100名 × 8,000円 × 12ヶ月 = 960万円
成果報酬：960万円 × 20% = 192万円
```

### Phase 3: 全国・高度化（1年以降）

- **統合サービス**: カスタマーサクセス全般
