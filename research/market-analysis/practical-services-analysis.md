## 1️⃣ 防災グッズ定期更新サービス

### 市場規模・成長性
- **防災食品市場**: 312.8億円（2021年）→ 319.1億円（2026年予測）
- **防災用品全体**: 3,500-4,000億円（2024年、2018年比20-30%増）
- **賞味期限**: 3-5年サイクルでの更新需要

### なぜお金を払うのか？
#### 明確な価値提供
1. **忘却リスクの回避**：賞味期限切れ = 非常時に使えない = 生命リスク
2. **選定の専門性**：何をどれだけ備蓄すべきか分からない
3. **継続管理の手間**：定期的なチェック・更新の負担
4. **法人需要**：オフィス防災の法的義務・BCP対策

#### 競合優位性
- **公共サービスとの非競合**：自治体は備蓄を管理してくれない
- **差別化しやすい**：地域特性（九州なら台風）、業界特性（オフィス環境）
- **既存強みとの親和性**：通販ノウハウ、在庫管理、コールセンター

### サービスモデル案
```
Phase 1: 個人向け「防災BOX定期便」
- 年1回の防災用品チェック・更新サービス
- 地域災害リスクに応じたカスタマイズ
- 3-5年計画での段階的入れ替え

Phase 2: 法人向け「BCP防災管理」
- オフィス・店舗の防災用品一括管理
- 法的要件チェック・更新代行
- 災害時対応マニュアル付き

Phase 3: 地域密着特化
- 九州・台風対策特化サービス
- 自治体・町内会向け一括サービス
```

### 収益モデル
- **商品売上**: 防災グッズの販売マージン
- **管理手数料**: 年間管理料（商品代の10-20%）
- **付加サービス**: 災害時サポート、講習会等

## 2️⃣ 更新アウトバウンド代行サービス

### 市場背景
- **サブスク市場の急拡大**: 継続課金モデルの普及
- **継続率の重要性**: 96%継続率が業界標準、1%向上で大きな売上インパクト
- **専門性の必要**: 効果的な更新アプローチは専門ノウハウが必要

### なぜお金を払うのか？
#### B2B明確価値
1. **売上直結**: 継続率1%向上 = 年間売上数百万〜数千万円増
2. **専門性**: 効果的な更新アプローチの研究・実績
3. **工数削減**: 自社スタッフのコア業務集中
4. **成果保証**: 継続率改善の数値コミット

#### 対象市場
- **年間パス系**: ジム、習い事、サブスク全般
- **保険業界**: 自動車保険、生命保険の更新
- **会員制サービス**: クレジットカード、各種会員サービス
- **SaaSサービス**: 法人向けシステムの継続契約

### サービスモデル案
```
Phase 1: 小規模サブスク企業向け
- 継続率90%未満の企業をターゲット
- 成果報酬型（継続率改善分の一定%）
- 更新前30-60-90日の段階的アプローチ

Phase 2: 業界特化展開
- フィットネス業界特化
- 教育・習い事業界特化
- それぞれの業界特性に応じたアプローチ

Phase 3: 包括的カスタマーサクセス
- 更新だけでなく、LTV最大化まで支援
- データ分析・改善提案込み
```

### 収益モデル
- **成果報酬**: 継続率改善分の20-30%を1年間
- **固定費**: 月額基本料金 + 成果報酬
- **コンサル**: 更新戦略設計・改善提案
