# Northstar Evolution Log

## プロジェクト目的

## 変遷記録

### 2026-03-11: 初期セッション

**出発点の表現**:
```
- 働くことを楽しめる人を増やす
- 人の力を引き出せる人になる
- ココシスグループがこの業界に来てくれてよかったと思ってもらえるような人を輩出する
- 志高い大人を生み出す
```

**問題認識**:
「きれいな言葉すぎて自分のミッション・ボイス的なものと紐づいていない感覚」

**体験の深掘り結果**:
- 組織の構造による被害者→加害者への気づき体験が核心
- 個人の尊厳欠如への実体験
- 「OSとは人間がもともと持っている良心を、前面に出し続けるための習慣と仕組み」

**最終到達点**:
```
「人の尊厳を取り戻せる組織OSを作り、広げる」
```

**実現戦略の転換**:
- プロダクトアウト → マーケットイン
- 理念先行 → ビジネス構築力先行
- 壮大な目標 → 小さく始める実験
