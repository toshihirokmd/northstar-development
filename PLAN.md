# Northstar Development - 実行計画

## プロジェクト目標

**最終目標**: 「人の尊厳を取り戻せる組織OSを作り、広げる」の実現
**当面目標**: ビジネス構築力を獲得し、確実な事業基盤を築く

## フェーズ別計画

### Phase 1: 基盤構築（2026年3-6月）

#### 1.1 商材選定スキル習得
**目標**: マーケットを見て自社ビジネスとつなげるスキル獲得

**アクション**:
- [ ] 商材選定フレームワークの調査・学習
- [ ] 競合分析手法の習得
- [ ] マーケットリサーチ手法の実践
- [ ] 既存顧客データ分析（ニーズ抽出）

**成果物**:
- 商材選定チェックリスト
- マーケット分析テンプレート
- 顧客ニーズマップ

#### 1.2 コールセンター軸ビジネス実験
**目標**: 既存資源を活用した新サービス検証

**候補サービス**:
1. 見守りサービス（高齢者向け）
2. 近隣トラブル・クレーム解消サービス
3. 中小企業向けカスタマーサポート代行
4. オンライン秘書・事務代行

**実験計画**:
- [ ] 各サービスの市場調査
- [ ] 既存顧客への需要アンケート
- [ ] 1サービス選定して1ヶ月間試験運用
- [ ] 結果分析と改善

#### 1.3 学習・研究活動
**目標**: ビジネス構築の基礎力強化

**学習領域**:
- [ ] 事業計画策定
- [ ] 収益モデル設計
- [ ] 顧客開発手法
- [ ] マーケティング戦略

### Phase 2: 実証・拡大（2026年7-12月）

#### 2.1 事業モデル確立
**目標**: 収益性のある事業モデル構築

**アクション**:
- [ ] Phase1実験結果を基にした事業モデル設計
- [ ] 収益計画策定
- [ ] 運用体制構築
- [ ] 本格的サービス開始

#### 2.2 組織OS要素の導入
**目標**: ビジネス成功を基盤とした理念実現開始

**アクション**:
- [ ] 事業運営に「愛と仁」の要素を組み込む
- [ ] スタッフの尊厳を重視した組織運営
- [ ] 顧客との関係性に「相互理解と慈しみ」を導入
- [ ] 成功事例のドキュメント化

### Phase 3: 理念拡大（2027年以降）

#### 3.1 ノウハウ体系化
**目標**: 「組織OS」の再現可能な仕組み化

**アクション**:
- [ ] 成功パターンのフレームワーク化
- [ ] 教育・コンサルティングプログラム開発
- [ ] 他組織への導入支援開始

#### 3.2 影響力拡大
**目標**: 業界・社会への波及

**アクション**:
- [ ] 成功事例の発信（書籍・講演等）
- [ ] 同志ネットワークの構築
- [ ] 業界標準への影響

## リソース計画

### 人的リソース
- **コア**: としひろ（プロジェクトリーダー）
- **協力**: コールセンタースタッフ（90名の一部）
- **外部**: 必要に応じて専門家・コンサルタント

### 資金計画
- **Phase 1**: 学習・実験費用（50-100万円程度）
- **Phase 2**: 事業立ち上げ費用（500-1000万円程度）
- **Phase 3**: 拡大投資（1000万円以上）

### 時間配分
- **平日**: 業務時間の20-30%をプロジェクトに
- **休日**: 月4-8時間の集中作業時間
- **来期**: より多くの時間を「人に対するエネルギー」に投入

## リスク管理

### 主要リスク
1. **フォーカス分散**: 理想追求により現実的進捗が停滞
2. **エネルギー低下**: 現状維持に安住してしまう
3. **資金不足**: 投資タイミングでの資金調達失敗

### 対策
1. **月次振り返り**: 進捗とフォーカス状況の確認
2. **外部圧力**: 定期的な発信・報告による適度なストレス
3. **段階的投資**: 小さく始めて成功を確認してから拡大
